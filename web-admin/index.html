Admin Access

Sign in to manage your tenant. Deployed on Netlify with Identity.

1

Enable Netlify Identity

In your Netlify site (`Site settings → Identity`), click "Enable Identity".

  1. Set Registration to "Invite only"
  2. Invite yourself and client users
  3. Assign a tenant role (e.g., tenant:acme) or set app_metadata.tenant
2

Connect GitHub Storage

Create a GitHub Personal Access Token and connect your repo for storing tenant JSON files:

  1. Create a token with repo scope
  2. In Netlify → Site settings → Environment variables, add:
    • GITHUB_REPO = harbortech7/castgrid
    • GITHUB_BRANCH = main (optional)
    • GITHUB_TOKEN = your token
  3. Click "Deploy" to apply settings
3
